## Editing Notes

- For a strict three-minute cut, combine the final two rows and reduce the preflight hold.
- Use the Home arrival only briefly. The demo must become the protagonist.
- Avoid fast pans over small text. If text matters, hold long enough to read it.
- Do not show developer-only diagnostics unless a technical judge asks for them.
- Use `docs/media/*.png` exhibits as fallback visuals when a live screen is too dense for the recording.
