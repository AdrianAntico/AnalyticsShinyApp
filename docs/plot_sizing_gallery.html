Plot Sizing QA Gallery

Generated by qa_plot_sizing_gallery(). Every supported plot is built through its production AutoPlots function and captured through AutoQuant::ObjectToPNG, the same screenshot helper used by artifact generation.

Manual QA criteria
  • Are all axis labels visible?
  • Are category labels truncated?
  • Are legends clipped?
  • Are titles readable?
  • Are margins sufficient?
  • Are heatmap cells distinguishable?
  • Are tick labels overlapping?
  • Does the aspect ratio feel appropriate?
  • Are plots wasting excessive whitespace?
