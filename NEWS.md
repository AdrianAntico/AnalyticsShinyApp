# AnalyticsShinyApp 1.0.0

## 1.0.0-buildweek

- Build Week release candidate for Analytics Workstation.
- Installable R package with `run_workstation()` and `launch_workstation()`.
- Package-installed application resources under `inst/app`.
- Windows installer and lightweight Electron launcher path.
- Deterministic Build Week demonstration with governed investigation, replay, report contracts, claim verification, and integrity review.
- Package distribution diagnostics through `qa_package_distribution()` and `workstation_diagnostics()`.
