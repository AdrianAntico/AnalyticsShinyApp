# Code of Conduct

Analytics Workstation contributors are expected to keep discussion professional, evidence-oriented, and constructive.

## Expected Behavior

- Be respectful and specific.
- Critique ideas and implementations, not people.
- Support important claims with evidence, reproduction steps, or code references.
- Assume good intent while still asking for clarity when something is unclear.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private information.
- Repeatedly derailing technical discussion.
- Introducing secrets, private data, or unsafe instructions into issues or pull requests.

## Reporting

If a problem occurs, open a private communication channel with the repository maintainer listed in `DESCRIPTION`.
