# AI-Native Analytical Systems

Working title:

```text
AI-Native Analytical Systems:
Designing Software That Reasons Over Evidence
```

This directory contains the living manuscript source for the book.

It is not the final rendered book. It is the canonical manuscript workspace from which later outputs may be derived:

- full book
- concise book
- website
- GPT knowledge base
- white papers
- conference talks
- executive briefings
- developer documentation
- research notebooks

The source material includes:

- conversations and design prompts
- architecture documents
- product vision documents
- git history and implementation milestones
- QA logs and validation results
- experiments and research studies
- API evolution
- UX evolution
- software engineering decisions
- unresolved questions
- future research plans

The manuscript follows the same philosophy as the Analytics Workstation artifact architecture:

```text
Truth
-> Knowledge Base
-> Representation
-> Delivery
```

The book is one representation. It is not the only source of truth.

The goal at this stage is to preserve, expand, synthesize, and clarify. Pruning and audience-specific compression happen later.
