# Raw Conversation Capture

This directory is reserved for raw or lightly normalized conversation exports that support the Analytics Workstation book corpus.

The goal is completeness first.

Expected future files may include:

```text
chatgpt_original_thread_export.md
chatgpt_followup_thread_export.md
codex_current_thread_export.md
codex_autoquant_origin_thread_export.md
codex_analytics_shinyapp_continuation_export.md
```

Raw exports should preserve:

- user prompts
- assistant responses
- thread ids when available
- dates when available
- repository context
- file attachments referenced by the prompt
- tool/output summaries when relevant

Do not prune raw exports. Derived Source Packs and manuscript chapters can be condensed later.
